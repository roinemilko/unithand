# unithand
A short python script for easy handling of dimensions.

# Function
Standard constants and units are used by their common abbreviations. Space is interpreted as multiplication. Write an expression and press enter, then provide the desired unit of measurement or leave empty for SI units.

# Usage
Clone repo by
```bash
git clone github.com/roinemilko/unithand
```
Start a python venv by running
```bash
python3 -m venv .venv
source .venv/bin/activate
```
Then run main.py
```bash
python3 main.py
```
